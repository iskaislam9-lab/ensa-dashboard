# ENSA Agadir — Dashboard des Tendances de Carrière

## Description
Ce dashboard interactif analyse les tendances de carrière et les choix de filières des étudiants de l'ENSA Agadir.

## Comment utiliser
1. Ouvrir le dashboard via le lien Streamlit
2. Utiliser le filtre dans la barre latérale pour filtrer par année d'études
3. Consulter les graphiques par section :
   - Statistiques globales (tous les étudiants)
   - Cycle Préparatoire (AP1/AP2)
   - Cycle Ingénieur (CI1)
4. Télécharger les données en CSV via le bouton de téléchargement

## Public cible
- Administration de l'ENSA Agadir
- Corps enseignant
- Clubs étudiants

## Technologies utilisées
- Python
- Streamlit
- Google Sheets API
